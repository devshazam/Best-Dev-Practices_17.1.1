# Server Action
	- Вызов через <form action={...}> с валидацией на стороне сервера
		- Передает объект formData в POST запросе с клиента на сервер
		- Передача дополнительных аргументов возможна при помощи метода bind()
	- Вызов через события (пример: onClick, onChange ...)
		- аргументы можно добавлять произвольно
